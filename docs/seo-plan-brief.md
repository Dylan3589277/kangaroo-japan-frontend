# 独立站SEO优化 — 需求补充：东南亚各国独立规划

## 补充说明

花哥强调：**东南亚各国不能只靠英文SEO混过去**，每个国家/地区需要单独的SEO策略。

## 重点市场

**优先覆盖前4个：泰国、越南、印尼、菲律宾**（人口多+日本商品受欢迎）

### 1. 泰国市场
- **语言**：新增泰语（th），内容/页面/搜索全泰语化
- **搜索优化**：Google Thailand + LINE搜索
- **结构化数据**：泰语Product schema
- **内容策略**：日本化妆品/动漫周边/二手奢侈品在泰国特别火
- **社交渠道**：LINE（泰国70%+用户用LINE）、Facebook Thailand

### 2. 越南市场
- **语言**：新增越南语（vi）
- **搜索优化**：Google Vietnam + Cốc Cốc搜索引擎适配
- **内容策略**：日本電機製品/ゲーム/漫画在越南年轻人中流行
- **社交渠道**：Zalo

### 3. 印度尼西亚市场
- **语言**：新增印尼语（id）
- **搜索优化**：Google Indonesia
- **内容策略**：日本アニメ/コスメ/ファッション在印尼巨火
- **社交渠道**：Instagram/TikTok Shop

### 4. 菲律宾市场
- **语言**：英文为主（已覆盖），但内容需针对性
- **搜索优化**：Google Philippines（英文统一处理）
- **内容策略**：日本性价比商品、日本美食、技术产品

### 5. 马来西亚市场
- **语言**：英文+马来文+中文（英文已覆盖，中文已覆盖）
- **搜索优化**：Google Malaysia
- **内容策略**：日本代拍多语言商品对比

### 6. 新加坡市场
- **语言**：英文为主（已覆盖）
- **搜索优化**：Google Singapore
- **内容策略**：高端日本商品、奢侈品

## 需要设计方案的具体内容

### 技术层面

1. **新增语言支持**：th（泰语）、vi（越南语）、id（印尼语）
   - next-intl 多语言扩展方案
   - 翻译文件组织策略（先AI自动翻译核心页面，后续人工优化）
   - hreflang tags 扩展（th-th、vi-vn、id-id、en-ph、en-sg、en-my、ms-my）

2. **区域化SEO**
   - Google Search Console 多国家配置（每国一个property或带国家code的hreflang）
   - 国家特定站内页面
   - URL结构：/country-th、/country-vn 或继续用 [lang] 路由

3. **重点平台**
   - LINE搜索优化（泰国）
   - Cốc Cốc搜索引擎适配（越南）
   - YouTube/Google Images（所有市场）

### 内容策略

1. **泰国**：日本コスメ購入ガイド（泰语）、日本アニメグッズ代行
2. **越南**：Hướng dẫn mua hàng Nhật Bản（越南语）、日本ゲーム機最安値比較
3. **印尼**：Panduan Belanja Jepang（印尼语）、日本スニーカー代行
4. **菲律宾**：Best Japanese Deals for Filipinos（英语，但菲律宾本地化）

### 实施优先级

**第一层（最优先）**：泰语、印尼语 — 这两个市场日本商品需求最大
**第二层**：越南语 — 年轻人口多增长快
**第三层**：菲律宾/新加坡/马来西亚 — 英文已覆盖，优化内容即可
